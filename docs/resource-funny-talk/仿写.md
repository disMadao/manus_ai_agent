1. 通勤地铁把我挤成了消化系统的一部分
早晚高峰的十三号线，据说灵感来自沙丁鱼罐头，但沙丁鱼至少不用站着睡觉。我在人群里被夹成一片海苔，前后左右都是别人的腋下和背包拉链，终于明白了什么叫“众生皆苦，只有我是佐料”。

某天被人流推着走的时候突然想到，如果把通勤路线画成食物链，我应该在最底层——每天被消化一次，从回龙观站输送到西二旗，晚上再被反向排出来。只是不知道终点是哪个下水道。

最魔幻的是某次加班到深夜终于有座，对面大哥抱着电脑改PPT，边改边对着屏幕自言自语。我想起自己白天开会时也是这副德性，顿时理解了什么叫社畜的尽头是精神病。

下车时看了他一眼，他也抬头看了我一眼，眼神里写着同病相怜。但我们谁也没说话，毕竟i人在地铁里唯一的社交礼仪就是假装对方不存在。

2. 体检报告单比我的人生规划还厚
入职三年第一次认真看体检报告，发现除了头发量，其它指标都在稳步增长。尿酸高、血脂高、脂肪肝轻度，医生建议清淡饮食、规律作息、适度运动。

我盯着这六个字看了很久，想告诉他我每天的饮食清淡得像兔子，作息规律得像闹钟，运动量取决于电梯排队的时长。但想了想还是点点头说谢谢，毕竟医生的KPI是开建议，不是帮我实现。

回家路上买了盒护肝片，拍了张照发家族群。我妈秒回：少熬夜。我回：好的。然后继续熬夜。

第二天同事问我为什么黑眼圈更重了，我说护肝片可能过期了。他沉默了一会儿，问我是不是需要找人聊聊。我说不用，我只是在实践一种很新的养生——边作死边补救，让身体在希望和绝望之间反复横跳，据说能激活免疫系统。

他走开了。但我没说谎，百度上真是这么写的。

3. 周末是当代打工人的薛定谔状态
周五晚上是最幸福的时刻，因为感觉拥有整个周末。周六早上醒来发现已经十一点，幸福指数下降30%。下午躺床上刷手机到三点，开始焦虑还有什么事没做。等终于决定出门的时候太阳下山了，幸福指数清零并转入负债。

周日晚上是最痛苦的时刻，因为周一就像悬在头顶的达摩克利斯之剑。区别是达摩克利斯只需要担心一把剑，而我要担心七个会议、三个deadline和一个临时需求。

某次周日失眠到三点，干脆起来写了份辞职信，措辞真挚、情感充沛，把自己感动哭了。第二天早上挤地铁的时候拿出来看了看，觉得情绪价值已经到位了，于是删掉继续上班。

同事问我周末去哪了，我说在平行宇宙里辞职了。他说哦，然后继续改bug。

大概这就是成年人的默契——大家都知道彼此在假装活着，但谁也不会戳破。

4. 三十岁之前我要退休，三十岁之后我要活到退休
最近发现同龄人已经开始聊养老了。不是那种“我要环游世界”的养老，是“城乡居民养老保险哪个档位划算”的养老。

群里有人发了张截图，显示按当前缴纳基数，退休后每月能领3876元。有人算了笔账，说按现在的通胀，到时候大概够买半个月的菜。沉默持续了三十秒，然后有人发了个红包，备注：先充点。

抢到两毛三，觉得生活又有盼头了。

另一个朋友开始研究延迟退休政策，算了算自己可能要干到六十五。我说往好处想，可能活不到领养老金那天。他说那更亏了，交都交了。

我想了想，说要不你把这笔钱当成年金险加定期捐赠，虽然自己用不上，但能为社保基金做贡献，死后还能在统计数字里留下一个名字。他说明年扫墓的时候会记得告诉我一声结果。

挂了电话突然觉得，能这么坦然讨论死后的事，大概也是一种长大。只是不知道为什么，长大和变老听起来像两件事，过起来却像同一件。
